# 🎮 Arshia's Game Development Portfolio

## 🕹️ Games Included

| Game | Description | Status |
|------|-------------|--------|
| 🐍 **Snake** | Classic snake game. Eat food, grow longer, avoid walls and yourself. High scores saved in Local Storage. | ✅ Complete |
| 💣 **Minesweeper** | Classic minesweeper with custom board dimensions. Left-click to reveal, right-click to flag mines. | ✅ Complete |
| 🧩 **Falling Blocks** | Tetris-style puzzle game. Rotate and place blocks to clear rows and score points. | ✅ Complete |
| 🦘 **Doodle Jump** | Platform jumper. Jump on platforms to go higher. Don't fall off the screen. | ✅ Complete |
| 🏓 **Ping Pong AI** | Play against a computer AI opponent. Realistic ball physics and collision detection. | ✅ Complete |
| 🔲 **Dots and Boxes** | Turn-based strategy game. Connect dots to complete boxes. Claim the most boxes to win. | 🟡 In Progress |

## 🧠 What I Learned

Building these games helped me develop:
- **Problem-solving**: Debugging collision detection and game logic
- **Algorithm design**: Implementing flood fill (Minesweeper), AI pathfinding (Ping Pong), and rotation logic (Falling Blocks)
- **User experience**: Creating smooth, responsive gameplay
- **Version control**: Managing code with Git and GitHub
- **Deployment**: Publishing a live website with GitHub Pages
